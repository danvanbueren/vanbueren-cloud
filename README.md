# vanbueren-cloud

A simple quick-access link tree directing visitors to subdomains and external services hosted under the `vanbueren.cloud` domain.

## Project Structure

- `vanbueren-cloud/` - The Next.js application directory.
- `AGENTS.md` - AI developer guidelines and project rules.
- `.github/` - GitHub configurations (Dependabot, workflow tools).

## Getting Started

To run the application locally:

1. Navigate to the application folder:
   ```bash
   cd vanbueren-cloud
   ```
2. Install the dependencies:
   ```bash
   bun install
   ```
3. Start the local development server:
   ```bash
   bun run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- **Framework**: Next.js (App Router)
- **UI Library**: Material UI (MUI 9) with Emotion styling
- **Hosting & Analytics**: Optimized for Vercel with Vercel Analytics & Speed Insights
